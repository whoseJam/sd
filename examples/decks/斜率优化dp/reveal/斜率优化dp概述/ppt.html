斜率优化 dp

概述

典型形式

$$f_i = \min_{j < i} \{ f_j + A_i B_j \}$$

朴素 $O(n^2)$。

几何视角

把 $f_j + A_i B_j$ 看作直线 $y = A_i x + c$ 在 $x = B_j$ 处的取值 — 等价于"用斜率 $A_i$ 的线扫过点 $(B_j, f_j)$，取最低截距"。

三件事

  1. 从转移方程推出"点 + 直线截距"的形式
  2. 维护转移点的下凸包
  3. 用固定斜率的线在凸包上找切点

维护凸包

转移点 $x$ 单调 — 用栈维护下凸包，新加点 pop 掉违反凸性的栈顶。

$x$ 不单调 — 用平衡树（不在本节讨论）。

查询切点

下凸包斜率单调 — 给定斜率二分定位最优切点。

斜率也单调时省掉二分，用指针单调推进。
