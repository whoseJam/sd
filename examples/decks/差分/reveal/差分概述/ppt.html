差分

差分

$\{a\}=\{a_1, a_2, \dots, a_n\}$ 的差分数组 $\{d\}$ 定义为 $$d_i = a_i - a_{i-1}$$ 其中 $a_0$ 看作 $0$，所以 $d_1 = a_1$。

差分与前缀和

用 $\{d\}$ 表示 $\{a\}$ $$a_i = \sum_{k=1}^i d_k$$

  • 站在 $\{a\}$ 的角度，$\{d\}$ 是 $\{a\}$ 的差分
  • 站在 $\{d\}$ 的角度，$\{a\}$ 是 $\{d\}$ 的前缀和

差分对前缀和的影响

修改 $\{d\}$ 中一个元素，$\{a\}$ 中所有后续元素都跟着变化 — 这正是快速区间修改的基础。

形式化定义

差分 $D$ 和前缀和 $S$ 是一对互逆运算： $$\{d\} = D(\{a\}), \qquad \{a\} = S(\{d\})$$

它们都是 $O(n)$ 的。

$\{a\} = S(D(\{a\}))$。

序列加法

定义 $\{a\} + \{b\}$ 为 $c_i = a_i + b_i$。

区间加法 = 增量序列加法

对 $[l, r]$ 全体加 $k$，相当于加一个增量序列 $\{0,\dots,0,k,\dots,k,0,\dots,0\}$。

$D$ 对序列加法满足分配律

$D(\{a\} + \{b\}) = D(\{a\}) + D(\{b\})$

同样 $S(\{a\} + \{b\}) = S(\{a\}) + S(\{b\})$。

所以若 $\{a\} = \{c_1\} + \dots + \{c_m\}$，可以先对每个 $c_i$ 差分再求和再前缀和。

稀疏化

每个 $D(\{c_i\})$ 通常很稀疏 — 区间加法的 $D$ 只在两点非零。

时间复杂度被压缩到 $O(n + m)$。
