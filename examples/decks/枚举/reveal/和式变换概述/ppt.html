和式变换

线性性

$\sum$ 是线性的，求和顺序可以重排：

$$\sum_{i=1}^n (a_i + b_i) = \sum_{i=1}^n a_i + \sum_{i=1}^n b_i$$

常数提取

常数因子可以提到 $\sum$ 外：

$$\sum_{i=1}^n c\, a_i = c \sum_{i=1}^n a_i$$

独立求和的乘积

当下标互相独立，两个 $\sum$ 可以合并成乘积：

$$\sum_{i=1}^n a_i \sum_{j=1}^m b_j = \left(\sum_{i=1}^n a_i\right) \left(\sum_{j=1}^m b_j\right)$$

交换枚举顺序

同一个样本空间，行扫描和列扫描覆盖相同的格点：

$$\sum_{i=1}^n \sum_{j=1}^m f(i, j) = \sum_{j=1}^m \sum_{i=1}^n f(i, j)$$

交换枚举顺序 — 三角形

对约束 $j \le i$ 的三角区域：

$$\sum_{i=1}^n \sum_{j=1}^{i} = \sum_{j=1}^n \sum_{i=j}^n$$

交换枚举顺序 — 因子枚举

枚举 $(i, d)$，要求 $d \mid i$：

$$\sum_{i=1}^n \sum_{d \mid i} = \sum_{d=1}^n \sum_{i:\, d \mid i, i \le n}$$

列扫描下，每列正好是 $\lfloor n / d \rfloor$ 个倍数 — 比逐个 $i \% d$ 试更快。
