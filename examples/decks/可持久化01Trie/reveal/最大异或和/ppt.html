最大异或和

题意

  1. A x：序列末尾追加 $x$
  2. Q l r x：找 $p \in [l, r]$ 使 $a_p \oplus a_{p+1} \oplus \dots \oplus a_n \oplus x$ 最大

前缀异或

令 $\text{sum}_i = a_1 \oplus \dots \oplus a_i$，则 $a_p \oplus \dots \oplus a_n = \text{sum}_n \oplus \text{sum}_{p-1}$。

原式 $= (\text{sum}_n \oplus x) \oplus \text{sum}_{p-1} = C \oplus \text{sum}_{p-1}$。

即在 $\text{sum}_{l-1}, \dots, \text{sum}_{r-1}$ 中找与 $C$ 异或最大的 — 可持久化 01Trie 上做区间查询。
