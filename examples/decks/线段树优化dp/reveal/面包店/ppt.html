面包店

CF 833B

状态

$f_{i, j}$ = $[1, i]$ 分成 $j$ 段的最大价值，最后段是 $[x + 1, i]$：

$$f_{i, j} = \max_{x < i} \{f_{x, j-1} + \text{diff}(x + 1, i)\}$$

线段树维护

对每个 $j$ 一棵线段树，维护 $f_{x, j-1} + \text{diff}(x + 1, i)$ 关于 $x$ 的值。

$i \to i+1$ 时新加面包 $a_{i+1}$，设上一次出现位置 $\text{pre}$：

  • $x \in [1, \text{pre}]$：$\text{diff}$ 不变
  • $x \in [\text{pre} + 1, i]$：$\text{diff}$ 加 1（区间加 1）
