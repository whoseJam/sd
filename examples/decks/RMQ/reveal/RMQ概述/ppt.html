RMQ

Range Maximum / Minimum Query

区间最大值问题

给定数组 $\{A\}=A_1, \dots, A_n$，多次查询区间最大值 $$Q(l, r) = \max\{A_l, A_{l+1}, \dots, A_r\}$$

$m$ 次查询，总复杂度 $O(nm)$。

SparseTable (ST 表)

  • 预处理 $O(n \log n)$
  • 单次查询 $O(1)$
  • 空间 $O(n \log n)$

ST 表

预处理二维表 $F$：$F_{i,j} = $ 从 $A_i$ 起、长度为 $2^j$ 的区间最大值。

任意区间都可以拆为两个长度 $2^k$ 的区间的并 — 直接查 $F$。

ST 表预处理：第 0 行

$F_{i, 0} = A_i$。

ST 表预处理：第 $j$ 行

$F_{i, j} = \max(F_{i, j-1},\ F_{i + 2^{j-1}, j-1})$

ST 表查询

给定区间长度 $\text{len}$，取 $k = \lfloor \log_2 \text{len} \rfloor$： $$Q(l, r) = \max(F_{l, k},\ F_{r - 2^k + 1, k})$$

预处理 log₂

系统 $\log_2$ 慢。预处理表：$\text{Log}_2[i] = \text{Log}_2[i / 2] + 1$。
