单调队列优化 dp

适用 dp 形式

$$f(i) = \max_{i - t \le j \le i} \{ G(j) + H(i) \}$$

  • $G(j)$ 只跟 $j$ 有关
  • $H(i)$ 只跟 $i$ 有关

$H(i)$ 提到 $\max$ 外面：$f(i) = \left( \max_{i - t \le j \le i} G(j) \right) + H(i)$，化为滑动窗口最大值。

优劣性

窗口内若 $j_1 < j_2$ 且 $G(j_1) \le G(j_2)$，则 $j_1$ 永远不可能成为最优解。

再叠加 $j < i - t$ 的过期剔除，留下的 $j$ 形成单调递减队列。

dp 递推

$i$ 从 $1$ 扫到 $n$，窗口滑动，$f(i) = \max_{j \in \text{window}} G(j) + H(i)$。

每个 $j$ 入队、出队至多各一次，总复杂度 $O(n)$。
