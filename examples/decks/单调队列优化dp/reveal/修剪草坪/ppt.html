修剪草坪

luogu P2627

题意

$n$ 头牛，效率 $E_i$。连续工作的牛数不能超过 $K$。求最大效率之和。

状态定义

$g(i)$ = 安排完前 $i$ 头，第 $i$ 头不干活的最大效率。

$f(i)$ = 安排完前 $i$ 头，第 $i$ 头干活的最大效率。

$g(i) = \max\{f(i-1),\, g(i-1)\}$

$f(i) = \max_{i - K \le j < i}\{g(j) + S(i) - S(j)\} = S(i) + \max_{i - K \le j < i}\{g(j) - S(j)\}$

其中 $S(i) = \sum_{k=1}^{i} E_k$。

单调队列

对 $g(j) - S(j)$ 做窗口长度为 $K$ 的滑动最大值。

答案 $= \max\{f(n), g(n)\}$。
