# SD 动画框架

## 动画框架引入

使用一行代码 `import * as sd from "@/sd";`，即可完成引入

## 动画框架简介

SD 动画框架在 SVG 之上提供轻量级对象封装，将原始 SVG 元素映射为链式 JS 对象。以矩形为例：

```js
const svg = sd.svg(); // 1200×600 画布
const rect = new sd.Rect(); // 创建矩形
rect.setX(100).setWidth(30); // 链式设置属性
svg.appendChild(rect); // 将 rect 添加到画布上
```

读取属性时，调用 get 方法：

```js
console.log(rect.getX()); // 100
```

你可以在创建对象时，使用 config 字典快速完成属性设置：

```js
const svg = sd.svg();
const rect = new sd.Rect({
  targetNode: svg,
  x: 100,
  y: 100,
  width: 30,
  height: 60,
}); // 无需调用 svg.appendChild(rect) 了
```

# 动画

动画框架提供关键帧动画和逐帧动画，这两种动画的制作，并且它们是互斥的，选了一个就不能用另一个了

## 关键帧动画

一般使用 `sd.main` 来引导关键帧动画：

```ts
sd.main(async () => {
  await sd.pause(); // 暂停，直到用户点击 'N' 键才继续
  rect.startAnimate().setX(100).setY(100).endAnimate(); // 创建一个关键帧动画，对 x 和 y 并行进行关键帧动画，使用默认配置（delay = 0ms，duration = 300ms）
  await sd.pause(300); // 暂停 300ms
  rect.startAnimate().setWidth(100).setHeight(100).endAnimate(); // 创建一个关键帧动画，对 width 和 height 并行进行关键帧动画，使用默认配置
});
```

把属性修改置于 `startAnimate` / `endAnimate` 之间即可生成关键帧动画：

```ts
rect.startAnimate({
    delay: 0,
    duration: 300,
    easing: // 指定缓动函数，可以是字符串，或者是一个把 [0, 1] 映射到 [0, 1] 的函数对象
}) // 0 ms 开始，持续 300 ms
    .setX(100) // 目标值
    .setWidth(30)
    .endAnimate();
```

多段动画按声明顺序依次执行：

```ts
rect
  .startAnimate({
    delay: 0,
    duration: 300,
  })
  .setX(100)
  .endAnimate(); // [0, 200]
rect
  .startAnimate({
    delay: 200,
    duration: 500,
  })
  .setX(200)
  .endAnimate(); // [200, 700]
```

注意，禁止对一个属性做时间线上有重合的关键帧动画，这种行为的结果是无法预测的

默认动画时长 300 ms；如无特殊需求，优先使用默认值

使用 `sd.pause` 进行暂停，可以选择手动设置暂停多少毫秒，也可以选择暂停直到用户点击 'N' 以继续

```ts
await sd.pause(1500); // 暂停 1500ms
await sd.pause(); // 暂停，直到用户点击 `N` 以继续
```

如果选择了暂停的时间，你应该保证该暂停之前的动画的执行是长于该暂停时间的，例如：

```ts
lineA.startAnimate({ duration: 600 }).setX2(B.x).setY2(B.y).endAnimate();
lineB.startAnimate({ delay: 200, duration: 600 }).setY2(A.y).endAnimate();
lineC
  .startAnimate({ delay: 400, duration: 600 })
  .setX2(B.x)
  .setY2(B.y)
  .endAnimate();
await sd.pause(1000); // 暂停 1000ms，前面的执行时间最长的对象是 lineC，正好执行了 1000ms，没有超过暂停时间
```

关键帧动画在框架下，是自动可逆的

## 逐帧动画

使用 sd.loopUpdate 来完成逐帧动画，逐帧动画不可使用 startAnimate 和 endAnimate 机制，直接开始 set 各种属性就行了，sd.loopUpdate 类似于 JS 中的 requestAnimationFrame，不过每次 update 之后不需要重新调用自身

```ts
sd.loopUpdate((t: number) => {
  // 逐帧动画更新
  // t 是一个从 0 开始的时间，单位是毫秒
  // 该函数每一帧都会调用一次
});
```

## 画布与坐标

- 固定尺寸：1200 × 600 px，不可修改
- 左上角 (0, 0)，右下角 (1200, 600)
- 元素尽量居中，避免遮挡，保持版式整洁

## 代码规范

使用 const 替换 let，尽量让所有的变量都使用 const 定义，const 优先

## 可用对象速览

一些常见的可调节参数，几乎对所有的有形状的元素都有效果：transformOrigin，translate，rotate，scale，opacity，fill，fillOpacity，stroke，strokeOpacity，strokeWidth，strokeDashArray，strokeDashOffset，strokeLineCap，strokeLineJoin

关于这里的 transformOrigin，和传统的 svg 中默认 transformOrigin 位于 (0, 0) 所不同，我们这里的 transformOrigin 默认是位于元素的 (center, center) 的（即，默认为包围盒中心）

基本上所有的滤镜元素，都有这些可调节参数：in，in2，result，colorInterpolationFilters

但是只有 Filter 可以调节：x，y，width，height

### Text

可调节参数：x，y，fontSize，text

- fontFamily：目前只能选择 "Times New Roman" 或者 "Arial"，默认是 "Times New Roman"
- width：文本算出的宽度
- height：文本算出的高度
- centerX：被定义为 x + width / 2
- centerY：被定义为 y + height / 2
- maxX：被定义为 x + width
- maxY：被定义为 y + height

使用 `text.startAnimate({ duration: 500 }).setText("hello").endAnimate()` 的时候，动画框架会自动生成字形的插值动画，平滑地进行字形过渡，这提供了除了调节文字的透明度用于切换文本之外的、另一种调节文本内容的方案

### Math

当想要使用数学公式的时候，务必使用 sd.Math，而不是 sd.Text

可调节参数：x，y，fontSize，centerX，centerY，maxX，maxY

- text：数学公式，例如 a^2、\sum\_{i=1}^na_i
- width：数学公式算出的宽度
- height：数学公式算出的高度

使用 `math.startAnimate({ duration: 500 }).setText("a^2").endAnimate()` 的时候，动画框架会自动生成字形的插值动画，平滑地进行字形过渡，这提供了除了调节透明度用于切换数学公式之外的、另一种调节数学公式内容的方案

### Rect

可调节参数：x，y，width，height，rx，ry，centerX，centerY，maxX，maxY

### Circle

可调节参数：cx，cy，r

- x：被定义为 cx - r
- y：被定义为 cy - r
- maxX：被定义为 cx + r
- maxY：被定义为 cy + r
- width：被定义为 2r
- height：被定义为 2r

### Ellipse

可调节参数：cx，cy，rx，ry，x，y，maxX，maxY，width，height

### Image

可调节参数：src

### Line

可调节参数：x1，y1，x2，y2，x，y，width，height

### Path

可调节参数：d

### Group

用来容纳其他元素

### Filter

用来容纳一个 filter 组合的，其内部可以 appendChild

可调节参数：id，x，y，width，height

### GaussianBlur

可调节参数：stdDeviation

### Offset

可调节参数：dx，dy

### DropShadow

可调节参数：stdDeviation，dx，dy，floodColor，floodOpacity

### ColorMatrix

可调节参数：stdDeviation，type，values

### Composite

可调节参数：operator，k1，k2，k3，k4

### Blend

可调节参数：mode

## 布局说明

布局顺序：先确定尺寸，再定位中心，避免二次偏移。

```js
rect.setWidth(50).setCenterX(80); // 中心精准落在 80
```

对于文字类的布局，它的锚点位于文字的左上角

例如一个字符串 a，它的 x，y 坐标为 (100, 100)，中心位于 (120, 120)

字符串被修改之后变成了 aaa，它的 x，y 坐标仍然为 (100, 100)，不过由于文本变长了，宽度变宽了，所以中心坐标就变成了 (160, 120)

如果你希望文本的中心一直位于某个点上，当修改了文本内容之后，需要重新设置 cx，cy 进行定位

```js
text.setText("world"); // 修改文本内容
text.setCx(point[0]).setCy(point[1]); // 重新设置文本的中心
// text.setCenter(point); // 或者使用这种简化写法
```

## 动画风格指南

- 像播放一段完整视频，节奏流畅、画面精美、知识准确
- 采用和谐浅色配色，元素丰富，版式平衡，杜绝遮挡
- 优先使用浮现、生长、强调，字形渐变等微动效，避免生硬跳变
